What is Stack Kitchen?

Stack Kitchen is like an evening cookery course in programming.

The idea is, you come along to an evening, bring your laptop, and you learn how to get cooking in one modern programming stack a month. Just like you'd learn one recipe a month.

There'll be a few programming experts at each evening event. So you won't feel lost. But don't worry about feeling out of your depth either. If you haven't coded much before but you know you want to get going with something practical then Stack Kitchen is for you.

So whether this is your first time getting into programming, or whether you are returning to programming after years away, Stack Kitchen can help you get going with the current trendy recipes.

nodejs @ the Stack Kitchen

September 4th, 2013 @ ThoughtWorks, Holborn, London

Dubbed the lean stack because of it's association with lean startups, nodejs and mongodb are a very fast way to get up and going with web programming.

We'll show you how to install nodejs and mongodb on your Ubuntu or AppleMac laptop (Windows users, click here) and we can show you how to start a nodejs project and what sort of tools you need to do it.

Help! I've only got a Windows laptop

Poor you! Never mind, you can still come and try stuff out. All you need to do to be productive is to download VirtualBox and an Ubuntu operating system to run in it.
